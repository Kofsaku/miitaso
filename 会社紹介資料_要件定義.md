# 会社紹介資料 要件定義書

## 1. 資料の基本仕様

| 項目 | 仕様 |
|------|------|
| 形式 | PDF（横長スライド形式） |
| スライドサイズ | 16:9（1920×1080px） |
| 総ページ数目安 | 15〜20ページ |
| 用途 | 営業・採用・提携先への会社紹介 |
| 配布方法 | メール添付、画面共有、印刷 |

---

## 4. レイアウトルール

### グリッド・余白

| 要素 | 値 |
|------|-----|
| 上下マージン | 80px |
| 左右マージン | 100px |
| コンテンツ領域 | 1720×920px |
| カラム間ギャップ | 40〜60px |
| 要素間の余白 | 24〜32px |

### スライドレイアウトパターン

| パターン | 構成 | 使用場面 |
|---------|------|---------|
| **表紙型** | 全面グラデーション背景 + 中央テキスト | 表紙、セクション区切り |
| **タイトル+本文** | 上部見出し + 下部テキスト/図 | 説明スライド |
| **3カラム** | 見出し + 3つのカード並列 | サービス、強み紹介 |
| **2カラム** | 左テキスト + 右ビジュアル | 概要説明 |
| **数値強調** | 大きな数字 + 補足テキスト | 実績紹介 |
| **プロセス型** | 横並び4ステップ + 矢印接続 | 開発フロー |
| **表型** | 見出し + テーブル | 会社概要、比較 |

### 共通デザイン要素

| 要素 | スタイル |
|------|---------|
| カード | 角丸16px、白背景、影 `0 4px 24px rgba(0,0,0,0.06)` |
| アイコンバッジ | 56×56px、角丸12px、`Sky100→Blue100` グラデーション背景 |
| セクションラベル | `Sky 600` テキスト、見出しの上に小さく配置 |
| 区切り線 | 1px `Slate 100`、横幅100% |
| ページ番号 | 右下、`Slate 400`、12px |
| ロゴ | 各スライド右上 or 左下（小さく） |

## 5. スライド構成と必要コンテンツ

### 第1部：導入（3ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 1 | **表紙** | 全面グラデーション背景 | 会社名、ロゴ、キャッチコピー、日付（任意） |
| 2 | **目次** | タイトル+リスト | セクション一覧（5〜6項目） |
| 3 | **会社ミッション** | 中央テキスト（大きく） | ミッション文（20〜40文字）、補足説明（50〜80文字） |

### 第2部：会社概要（3ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 4 | **会社概要** | 表型 | 会社名、設立、代表者、所在地、従業員数、資本金、事業内容 |
| 5 | **沿革・実績** | タイムライン型 | 設立〜現在の主要イベント（3〜6項目） |
| 6 | **代表メッセージ** | 2カラム（写真+テキスト） | 代表写真、メッセージ文（150〜300文字）、氏名・肩書 |

### 第3部：事業内容（4〜5ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 7 | **解決する課題** | 3カラム（カード） | 課題3つ（アイコン+見出し+説明 各40〜60文字） |
| 8 | **サービス概要** | 3カラム（カード） | サービス3つ（アイコン+見出し+説明+特徴3〜5項目） |
| 9 | **サービス詳細1** | 2カラム | サービス1の詳細説明（100〜200文字）+イメージ図 |
| 10 | **サービス詳細2** | 2カラム | サービス2の詳細説明 |
| 11 | **サービス詳細3** | 2カラム | サービス3の詳細説明 |

### 第4部：強み・差別化（3ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 12 | **選ばれる理由** | 3カラム（数値強調カード） | 強み3つ（数値+見出し+説明 各60〜100文字） |
| 13 | **開発プロセス** | プロセス型（4ステップ横並び） | ステップ4つ（番号+アイコン+名称+説明 各40〜60文字） |
| 14 | **技術スタック** | アイコングリッド | 使用技術一覧（ロゴ+名称）、カテゴリ分け |

### 第5部：実績（2〜3ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 15 | **実績サマリー** | 数値強調（3〜4つ） | プロジェクト数、クライアント数、技術領域数など |
| 16 | **事例紹介1** | 2カラム | プロジェクト名、課題、解決策、成果、スクリーンショット |
| 17 | **事例紹介2** | 2カラム | 同上（別案件） |

### 第6部：クロージング（2〜3ページ）

| # | スライド | レイアウト | 必要コンテンツ |
|---|---------|-----------|-------------|
| 18 | **料金目安** | 表型 or カード型 | プラン別の価格帯、含まれるサービス |
| 19 | **お問い合わせ** | 中央テキスト | 連絡先（メール、電話）、URL、QRコード（任意） |
| 20 | **裏表紙** | 全面グラデーション背景 | ロゴ、会社名、「Thank you」 |

## 7. デザインの具体的な指針

### 表紙スライド

```
背景:    線形グラデーション 135deg, #0EA5E9 → #2563EB
装飾:    半透明の円形Blob（白 opacity 10〜15%）2〜3個配置
ロゴ:    白抜き版、中央上部
タイトル: 白、60〜72px Bold、中央配置
サブ:    白 opacity 80%、24px Regular、タイトル下
```

### 標準コンテンツスライド

```
背景:    #FFFFFF
左上:    セクションラベル（#0284C7、14px Semibold）
見出し:  #0F172A、36px Bold、ラベル直下
本文:    #475569、18px Regular
右上:    ロゴ小（グレースケール or カラー）
右下:    ページ番号（#94A3B8、12px）
下部:    1pxライン #F1F5F9
```

### カード

```
背景:      #FFFFFF
角丸:      16px
パディング: 32px
影:        0 4px 24px rgba(0,0,0,0.06)
ボーダー:   なし or 1px #E2E8F0
アイコン:   56×56 角丸12px Sky100→Blue100グラデ背景 + #0284C7アイコン
見出し:    #0F172A、20px Semibold
本文:      #475569、16px Regular
```

### セクション区切りスライド

```
背景:  #F8FAFC
中央:  セクション番号（#0EA5E9、64px Bold）
下:    セクション名（#0F172A、40px Bold）
装飾:  左に4px幅の #0EA5E9 縦ライン
```
