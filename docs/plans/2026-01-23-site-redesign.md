# miitaso サイトリニューアル設計書

## 概要

現在のサイトをプロフェッショナルな1ページLPに作り替える。

## 目的

- 新規事業相談・アプリ開発依頼の獲得
- スタートアップ・個人をターゲットに訴求

## デザイン方針

### カラーパレット

| 用途 | カラー | コード |
|------|--------|--------|
| 背景（メイン） | 白 | `#ffffff` |
| 背景（セクション） | 薄いグレー | `#f8fafc` |
| テキスト（メイン） | 紺 | `#0f172a` |
| テキスト（サブ） | グレー | `#64748b` |
| アクセント（明） | 水色 | `#0ea5e9` |
| アクセント（暗） | 青 | `#3b82f6` |
| 強調 | 紺 | `#1e3a5f` |

### デザインスタイル

- ライトモードベース
- 青〜水色のグラデーションをアクセントに
- 余白を大きくとったクリーンなレイアウト
- スクロールに応じたフェードインアニメーション
- Linear、Stripe 風のモダンな印象

## ページ構成（1ページLP）

### 1. ヒーロー

**キャッチコピー:**
> アイデアを、動くプロダクトに。

**サブコピー:**
> 新規事業の立ち上げからアプリ開発まで。
> 企画・設計・開発・運用をワンストップでサポートします。

**CTA:**
- メイン: 「無料で相談する」
- サブ: 「サービスを見る」

**背景:** 白 + 青系グラデーションのblob

### 2. 課題提起

ターゲットの悩みを提示:
- アイデアはあるが、どう形にすればいいか分からない
- 開発を依頼したいが、どこに頼めばいいか分からない
- 予算を抑えつつ、質の高いものを作りたい

### 3. サービス紹介

3つのサービスをシンプルに:
1. **新規事業サポート** - アイデアの整理から事業計画まで
2. **アプリ・Web開発** - 企画・設計・開発をワンストップで
3. **運用・改善** - リリース後の保守・グロースも支援

### 4. 選ばれる理由（強み）

3つの強みを訴求:
1. **一貫サポート** - 相談から運用まで一人の担当が伴走
2. **技術力** - 最新技術で高品質なプロダクトを実現
3. **コストパフォーマンス** - 無駄を省いた適正価格

### 5. 開発の流れ

4ステップで分かりやすく:
1. ヒアリング - 課題やアイデアをお聞きします
2. 企画・設計 - 要件定義とUI/UX設計
3. 開発 - アジャイル開発で柔軟に対応
4. 納品・サポート - リリース後も継続サポート

### 6. よくある質問（FAQ）

- 費用の目安は？
- 開発期間はどのくらい？
- どんな技術を使う？
- 相談だけでも大丈夫？

### 7. お問い合わせ

シンプルなフォーム:
- お名前
- メールアドレス
- ご相談内容

## 技術仕様

### 削除するもの

- `/blog` 関連ページ・コンポーネント
- `/admin` 管理画面
- `/tools` AIツール
- `/case-studies` 実績ページ
- `/services`, `/pricing`, `/about`, `/estimate`, `/faq`, `/resources`, `/cases` など個別ページ
- 認証機能（NextAuth）
- Prisma / データベース接続

### 維持するもの

- Next.js 15 (App Router)
- Tailwind CSS
- shadcn/ui コンポーネント
- お問い合わせフォーム + reCAPTCHA
- レスポンシブデザイン

### 新規追加

- スクロールアニメーション（Intersection Observer）
- グラデーション背景コンポーネント
- FAQアコーディオン
